Move battleBot folder to "Program Files (x86)\Arduino\libraries", include it by adding:

#include <battleBotSetup.h>
BattleBotSetup bb;

methods get called by bb.methodName

y = navigate to next challenge
u = follow the tape challenge
i = navigate out the maze challenge
o = pop the balloon challenge
p = Stop challenge

w = forward
a = left
s = backwards
d = right
q = speed to 0
f = speed to 125
r = speed to 250
z = turn 180 degrees
